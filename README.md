## ğŸš€ Getting Started

### Play Online

Just visit **[pakbaz.github.io/bingo-app](https://pakbaz.github.io/bingo-app/)** and start playing!

### Run Locally

**Prerequisites:**
- [Node.js 22](https://nodejs.org/) or higher

```bash
# Clone the repository
git clone https://github.com/pakbaz/bingo-app.git
cd bingo-app

# Install dependencies

# Start development server

```

Visit `http://localhost:5173` to play!

### Build for Production

```bash

```

The built files will be in the `dist` directory.
